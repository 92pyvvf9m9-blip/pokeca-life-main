# Pokeca Life v1.24.0

## 抽選が勝手に集まるポケカ抽選アプリ

Pokeca Lifeは、ユーザーが抽選情報を登録する掲示板型サービスではありません。公式サイト、公式X、店舗サイト、店舗アプリ、Googleフォーム、LivePocketなどをアプリ側が巡回し、抽選情報を自動で発見・解析・統合・公開するWebアプリです。

### 中心機能

- 自動収集
- 自動解析
- 自動公開
- 自動通知

手動追加、URL追加、スクリーンショット追加は、取りこぼしや例外に対応する補助機能です。

### 掲載基準

候補は次の条件を満たした場合だけ公開します。

1. 商品カタログとの一致
2. 応募締切日の確定
3. 直接応募先URLまたは公式店舗告知の確認
4. 日付順序・曜日の整合
5. 重複情報の統合

### Discovery Engine

v1.24.0から、全国情報源DBと新着・更新検知の基盤を導入しました。設計と実装範囲は [`DISCOVERY_ENGINE.md`](./DISCOVERY_ENGINE.md) を参照してください。

### 公開内容

画面に表示するのは、抽選参加に必要な事実情報と直接応募先です。収集元URL、内部ルール、検証経路などのDiscovery情報は公開しません。応募条件の最終確認は、各応募先ページで行ってください。
