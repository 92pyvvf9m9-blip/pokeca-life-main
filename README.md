# Pokeca Life v1.14.0

## ポケカ抽選情報を、速く・正確に・見やすく

Pokeca Lifeは、複数の公開情報を独自の収集・照合フローで整理し、店舗名、対象商品、応募期限、結果発表、購入期間、応募先を一覧化するWebアプリです。

### 掲載基準

候補は次の条件を満たした場合だけ公開します。

1. 商品カタログとの一致
2. 応募締切日の確定
3. 直接応募先URLの確認
4. 日付順序・曜日の整合
5. 重複情報の統合

### 公開内容

画面に表示するのは、抽選参加に必要な事実情報と直接応募先です。第三者ページの説明文やレイアウトは転載しません。応募条件の最終確認は、各応募先ページで行ってください。
